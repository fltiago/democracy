# Eleições & Dados

Landing page sobre integridade eleitoral brasileira, apresentando análise estatística dos dados do TSE com foco nas eleições de 2026.

## Páginas

| Rota | Descrição |
|---|---|
| `/` | Versão resumida em linguagem acessível |
| `/artigo` | Texto integral do artigo original |

## Requisitos

- Node.js 18+
- npm

## Setup

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev
```

A aplicação estará disponível em `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```

## Deploy

O projeto está configurado para deploy automático no Vercel. Basta importar o repositório em [vercel.com/new](https://vercel.com/new) — o Vercel detecta o Next.js automaticamente.
